## Frontend Engineer Exam 實作紀錄 📝
demo 網址：https://frontend-exam-iota.vercel.app/

---
### 技術內容

此專案使用 React 進行開發，並使用 Vercel 部署。

- React
- Zustand
- Sass Module
- Material UI
- Axios
- React Slick
- Vercel

---

### 專案下載使用方式

下載後，安裝 node_modules `yarn install`，並使用 `yarn run start` 即可執行專案。
